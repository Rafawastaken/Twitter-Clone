# Twitter Clone

> Rafawastaken | January 2023

## About

Twitter clone using Django and React

## Technologies Used:

- Backend:
  - Django (version 2.2)
- Frontend:
  - React
  - Bootstrap 5.3
- REST Api
- Database:
  - Sqlite3

## Functionalities

1. Tweets:

   - Create:
     - Text
     - Image:
       - Media Storage
   - Delete
   - Retweet
   - Likes

2. Users:

   - Register
   - Login
   - Logout
   - Profile:
     - Image
     - Text
     - Follow
   - Feed:
     - User's feed
     - User + who they follow

3. Following / Followers

## How to run

1. Clone
2. Install dependencies:
   > python3 -m pip install -r requirements.txt
3. Run it:
   > python run.py
